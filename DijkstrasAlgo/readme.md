# Execution

To compile the bubblesort code run the bellow command in the Terminal

```
$ gcc Dijkstras.cpp -lstdc++ -o dijkstras.out
```

It will create an a.out file in the same folder. Run the file.

```
$ ./dijkstras.out
```
